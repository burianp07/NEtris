# NETRIS - unity hra
Engine: Unity v6000.3.4f1, C#<br>
Stav:<br>
popis: Jedná se o netradiční verzi klasického Tetrisu, která obrací základní princip hry. Hráč zde nesmí dokončit (spojit) žádný celý řádek, protože jeho uzavření znamená konec hry nebo výraznou penalizaci. Hlavním cílem je proto co nejdéle udržet herní plochu zaplněnou bez vytvoření kompletní linie, a přitom nasbírat co nejvíce bodů.

Body hráč získává za:

strategické umisťování dílků,

vytváření složitých tvarů a mezer,

přežitý čas nebo počet správně položených bloků,

případně za rizikové tahy těsně před dokončením řádku.

Hra klade důraz na plánování, prostorovou představivost a sebekontrolu, protože hráč musí neustále balancovat mezi efektivním zaplněním plochy a vyhýbáním se dokončení řádku. Oproti klasickému Tetrisu je tempo napínavější a chyba může přijít velmi nenápadně. Tento režim nabízí originální výzvu i pro zkušené hráče, kteří hledají nový způsob, jak si Tetris vyzkoušet.
